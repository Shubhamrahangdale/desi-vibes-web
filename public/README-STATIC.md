# EventMitra - Static HTML/CSS/JS Version

This is the static HTML/CSS/JS version of EventMitra. All backend functionality has been removed.

## Files Included

- **index.html** - Home page with hero, features, and event previews
- **events.html** - Events listing page with sample events
- **login.html** - Login page (form only, no backend)
- **register.html** - Registration page (form only, no backend)
- **about.html** - About us page
- **contact.html** - Contact form page (form only, no backend)
- **styles.css** - All styling with Indian-inspired design
- **robots.txt** - SEO configuration

## How to Use

1. **Open Locally**: Simply open any HTML file in your web browser
   - Double-click `index.html` to start
   - Navigate between pages using the navigation menu

2. **Deploy to Web Server**: Upload all files to any static web hosting:
   - GitHub Pages
   - Netlify
   - Vercel
   - Any traditional web host

3. **No Build Process Required**: These are pure HTML/CSS/JS files that run directly in the browser

## Features

✅ Responsive design for mobile, tablet, and desktop
✅ Indian-inspired color palette (saffron, navy, gold)
✅ Beautiful animations and transitions
✅ SEO-optimized with proper meta tags
✅ Accessible navigation
✅ Clean, semantic HTML

## Limitations

⚠️ **No Backend Functionality**
- Authentication (login/register) forms are static only
- Contact form doesn't actually send emails
- Event bookings are not functional
- All event data is hardcoded

⚠️ **To Add Backend:**
You would need to:
1. Set up a backend server (Node.js, PHP, Python, etc.)
2. Add form handling for submissions
3. Connect to a database
4. Implement authentication system
5. Add payment processing

## Customization

- Edit `styles.css` to change colors, fonts, or layout
- Modify HTML files to change content
- Add more pages by copying existing HTML structure
